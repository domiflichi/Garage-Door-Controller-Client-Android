This repository contains all the source files you can use to compile your own
Android APK file.

For your convenience, the app is free to download from Google Play at:
https://play.google.com/store/apps/details?id=com.domiflichi.garagedoorcontroller

For more information and instructions on how to set all of this up and how everything works
together, please visit the home page of the project on my website:
http://jamienerd.blogspot.com/2013/12/garage-door-controller-tutorial.html
